Control Panel

Overview

controlpanel.js is a library to handle GUIs to set parameters.

Structure

Each control is sepalated into a class. The basic structure of the classes are the same.

@function appendTo(DOM_Element)

This is a function to append a container to given HTML DOM Element.

@function getValue()

This is a function to get the current value set by user using the contorl.

@function setValue(value, isExecuted)

This is a function to set value by program. We can decide the callback functions are executed after the value setting.

@function addCallback(callback)

This is a function to add a callback to be executed when the value of the control is changed.

@function removeCallback(callback)

This is a function to remove callback which set by addCallback().
